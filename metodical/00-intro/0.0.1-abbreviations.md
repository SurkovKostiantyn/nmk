# ПЕРЕЛІК УМОВНИХ СКОРОЧЕНЬ

У цих методичних вказівках використовуються такі умовні скорочення та абревіатури:

- **API** (Application Programming Interface) — інтерфейс програмування застосунків.
- **CDN** (Content Delivery Network) — мережа доставки вмісту.
- **CI/CD** (Continuous Integration / Continuous Deployment) — безперервна інтеграція та безперервне розгортання.
- **CLI** (Command Line Interface) — інтерфейс командного рядка.
- **CRUD** (Create, Read, Update, Delete) — чотири базові функції управління даними.
- **CSS** (Cascading Style Sheets) — каскадні таблиці стилів.
- **DB** (Database) / БД (База даних) — структурований набір електронних даних.
- **DNS** (Domain Name System) — система доменних імен.
- **DOM** (Document Object Model) — об'єктна модель документа.
- **FTP/SFTP** (File Transfer Protocol / Secure FTP) — протокол (безпечного) передавання файлів.
- **GUI** (Graphical User Interface) — графічний інтерфейс користувача.
- **HTML** (HyperText Markup Language) — мова гіпертекстової розмітки.
- **HTTP/HTTPS** (HyperText Transfer Protocol / Secure) — протокол (безпечного) передавання гіпертексту.
- **IDE** (Integrated Development Environment) — інтегроване середовище розробки.
- **JSON** (JavaScript Object Notation) — текстовий формат обміну даними.
- **LTS** (Long-Term Support) — версія програмного забезпечення з довгостроковою підтримкою.
- **MVC** (Model-View-Controller) — шаблон архітектури програмного забезпечення.
- **OOP** (Object-Oriented Programming) / ООП (Об'єктно-орієнтоване програмування) — парадигма програмування.
- **OS** (Operating System) / ОС (Операційна система) — базовий комплекс програмного забезпечення.
- **PHP** (PHP: Hypertext Preprocessor) — скриптова мова програмування для бекенд-розробки.
- **PR/MR** (Pull Request / Merge Request) — запит на злиття змін до основної гілки репозиторію.
- **RDBMS** (Relational Database Management System) / СУБД (Система управління базами даних) — система управління реляційними базами даних.
- **REST** (Representational State Transfer) — архітектурний стиль для розробки веб-сервісів (API).
- **SPA** (Single-Page Application) — односторінковий веб-додаток.
- **SQL** (Structured Query Language) — мова структурованих запитів для роботи з реляційними базами даних.
- **SSH** (Secure Shell) — мережевий протокол для безпечного (шифрованого) доступу до сервісів віддалено.
- **UI/UX** (User Interface / User Experience) — інтерфейс користувача та досвід користувача.
- **URI/URL** (Uniform Resource Identifier / Locator) — уніфікований ідентифікатор / локатор (адреса) ресурсу.
- **VCS** (Version Control System) / СКВ (Система контролю версій) — система, що фіксує зміни у файлах із часом.
- **WSL** (Windows Subsystem for Linux) — підсистема Windows для Linux.
