# Скрипти для генерації PDF

Скрипт `build_pdfs.ps1` призначений для інкрементальної конвертації Markdown файлів у PDF.
Основні можливості скрипта:

- Шукає `*.md` файли в усіх підпапках проєкту.
- **Ігнорує** кореневі `.md` файли (наприклад, `readme.md` чи загальні `plan.md`).
- Зберігає структуру каталогів у папці `/pdf` (наприклад, `web_technology/lecture_01.md` -> `pdf/web_technology/lecture_01.pdf`).
- **Інкрементальна збірка:** конвертує лише ті файли, які були змінені або додані після останньої генерації PDF, суттєво заощаджуючи час.

### Генерація всіх оновлених PDF

```powershell
.\build_pdfs.ps1
```

### Примусова генерація одного файлу

```powershell
.\build_pdfs.ps1 -FilePath "web_technology/lecture_01.md"
```
