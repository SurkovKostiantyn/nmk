# ☁️ Рекомендовані медіаресурси: Хмарні технології

Цей документ містить підбірку корисних відео, інтерактивних платформ, подкастів та курсів для поглиблення знань з дисципліни "Хмарні технології". Ви можете використовувати його як додатковий довідник.

## 📺 YouTube-канали та Плейлисти (Must Watch)

### 🔥 Загальне розуміння та Архітектура

- [TechWorld with Nana](https://www.youtube.com/c/TechWorldwithNana) — один з найкращих каналів для розуміння DevOps, Docker, Kubernetes та загальних хмарних концепцій. Пояснено максимально структуровано і просто.
- [NetworkChuck](https://www.youtube.com/c/NetworkChuck) — дуже емоційне та цікаве подання складних технічних тем: що таке хмара, як працюють мережі, віртуальні машини та Docker.
- [Fireship](https://www.youtube.com/c/Fireship) — формат "технологія за 100 секунд". Відмінно підходить для швидкого базового огляду архітектур (AWS, Firebase, Docker).

### ☁️ Провайдери (AWS, Azure, GCP)

- [AWS Training Center](https://www.youtube.com/user/AWSTutorials) — офіційні вебінари та туторіали по базових сервісах.
- [Azure Friday](https://www.youtube.com/playlist?list=PLLasX02E8BPBxGWTILcs0hDZYH1YtOPd5) — короткі відео про нові фічі Azure від провідних інженерів Microsoft.
- [freeCodeCamp.org - AWS/Azure Tutorials](https://www.youtube.com/c/Freecodecamp) — тут зібрано безліч повних курсів (по 10+ годин) для отримання сертифікацій (наприклад, AWS Certified Cloud Practitioner).

## 🎮 Інтерактивні платформи (Навчись на практиці)

Замість того, щоб розгортати складні середовища локально, ідеально підходять готові "пісочниці" у браузері:

- [AWS Skill Builder](https://skillbuilder.aws/) — офіційна платформа з практичними квестами та симуляціями (включаючи рольову гру AWS Cloud Quest).
- [Microsoft Learn for Azure](https://learn.microsoft.com/azure) — покрокові модулі. Багато завдань автоматично створюють тимчасові ресурси (sandbox) Azure для виконання роботи без прив'язки банківської картки.
- [Killercoda](https://killercoda.com/) — безкоштовні інтерактивні середовища в браузері для практики Linux, Docker, та Kubernetes.

## 🛠 Контейнеризація та мікросервіси

- [Play with Docker](https://labs.play-with-docker.com/) — офіційна безкоштовна лабораторія Docker у браузері (сесія дається на 4 години, потребує лише реєстрації на Docker Hub).
- [Docker Tutorial for Beginners (Nana)](https://www.youtube.com/watch?v=3c-iBn73dDE) — воркшоп, з якого варто почати знайомство з контейнерами.

## 🎧 Подкасти (Слухати в дорозі)

- [Cloudcast](https://www.thecloudcast.net/) — щотижневе обговорення трендів, архітектури та новин у світі Cloud Computing / Serverless (англ).
- [Шоу "Пройди IT"](https://podcasts.apple.com/ua/podcast/пройди-it/id1552393282) — українські подкасти, де крім розробки часто обговорюють кар'єру та задачі фахівців з DevOps та Cloud.

## 📚 Студентські вигоди (Увага!)

Для виконання лабораторних робіт та власних експериментів обов'язково використовуйте свою університетську пошту (`.edu.ua`), щоб отримати ресурси безкоштовно:

- **[GitHub Student Developer Pack](https://education.github.com/pack)** — включає доступ до DigitalOcean ($200 кредитів), Azure (для студентів) та десятків інших крутих сервісів.
- **AWS Educate / AWS Academy** — безкоштовні ресурси та навчальні панелі спеціально для ВНЗ (без потреби додавати кредитну картку).
- **[Azure for Students](https://azure.microsoft.com/en-us/free/students/)** — $100 кредитів на хмарні експерименти від Microsoft, що поновлюються щороку.
