# Звіт про роботу науково-технічного гуртка «ETI Cloud Club»

## за 2024–2025 навчальний рік

**Економіко-технологічний інститут імені Роберта Ельворті**  
Циклова комісія інформаційних технологій

---

Науково-технічний гурток **«ETI Cloud Club»** у 2024–2025 навчальному році продовжив свою діяльність, зосереджуючись на розвитку практичних навичок студентів спеціальності 122 «Комп'ютерні науки» у сфері хмарних технологій. Основними напрямками роботи стали вивчення хмарних платформ, принципів побудови хмарної інфраструктури, безпеки та сучасних DevOps-практик.

Протягом вересня – травня 2025 року гурток регулярно проводив засідання у форматі практичних занять, семінарів та проєктної роботи. Учасники виконували індивідуальні та командні завдання, презентували власні розробки та обговорювали актуальні технології.

Через зимові канікули у січні 2025 року засідання гуртка не проводилися.

## Висновки

Учасники гуртка **«ETI Cloud Club»** протягом 2024–2025 навчального року набули практичних навичок роботи з хмарними платформами AWS та Azure, контейнеризації (Docker, Kubernetes), побудови CI/CD pipeline та основ хмарної безпеки. Реалізовані проєкти демонструють здатність студентів застосовувати хмарні технології для вирішення реальних задач.

У наступному навчальному році планується розширення тематики у бік мультихмарних архітектур, Infrastructure as Code (Terraform) та підготовки до галузевих сертифікацій.
